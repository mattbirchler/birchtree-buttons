Birchtree.css

Birchtree.css is an easy-to-use, basic CSS library you can use in just about any web project you're working on. This is a work in progress, and is being created by a team of one, so please bear with me as I build out this project. If you have feeback, please feel free to email mattbirchler@outlook.com or comment on the GitHub project and I'll be happy to help.
